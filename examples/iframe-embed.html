🎮 Agent Arcade — Embed Examples

Embed Agent Arcade in any website using a simple <iframe>. Pass config via query params.

Default (Light)

<iframe src="http://localhost:3000/embed?session=my-session"></iframe>

Dark Mode

<iframe src="http://localhost:3000/embed?session=my-session&dark=1"></iframe>

Full-width embed

Full Width · Retro Arcade Theme

<iframe src="http://localhost:3000/embed?session=my-session&dark=1&theme=retro-arcade" style="width:100%;height:500px"></iframe>
